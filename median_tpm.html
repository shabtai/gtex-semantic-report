3Structural findings

Each finding is a join/aggregation hazard that survives into the curated product — the same “one entity, two identifiers” shape seen in the raw annotation tables.

Non-unique key

Description

In table 'GENE_MEDIAN_TPM', 163 distinct values of 'Description' each map to multiple 'Name' values (example: '5_8S_rRNA' maps to 5 IDs) (no target column detected; weak tier -- higher false-positive risk). This is the 'system-split' pattern: the same human-readable label has different IDs per source system. Queries that filter by a single Name will silently miss data from other systems.

Remediation: Aggregate by 'Description' (or an upstream concept), not 'Name'. When joining downstream data, use 'Description' as the join key — or expand any Name filter to include every id that maps to the same label.

Repeated rows

Name

Column Name is highly unique, masking 1149 duplicate rows in the remaining columns; 1149 rows (2.0%) are duplicates when ignoring Name

Remediation: Check if Name is an auto-incrementing ID/index and deduplicate using subset=[...] excluding Name before counting or summing

Duplicate identifiers

Name

1257 rows are duplicates across every column except 'Name'. Likely a botched re-load where Name differs but the rest is unchanged.

Remediation: Inspect df[df.duplicated(subset=df.columns.drop('Name'), keep=False)]; decide whether to keep the latest or merge the Name column.

Duplicate identifiers

Name

The dataset includes 44 Ensembl IDs mapped to the pseudo-autosomal regions (PAR) of the Y chromosome, represented with a '_PAR_Y' suffix. These IDs are duplicates of their non-suffixed counterparts (e.g., 'ENSG00000169093.15' and 'ENSG00000169093.15_PAR_Y' both encode 'ASMTL'). If analysts strip version suffixes to join across transcriptomic datasets, these base IDs will duplicate, potentially leading to fan-out biases in gene-level expression sums or downstream multi-table joins.

Remediation: Handle PAR regions explicitly before joining or mapping to base IDs: either aggregate the expression values across the pseudo-autosomal duplicate pairs, or filter out the '_PAR_Y' specific rows if the analysis focuses strictly on non-redundant loci representation.

4Correctly not flagged

Domain-aware restraint: a generic detector proposed 11 columns as “zeros mean missing”. The verifier rejected all of them — in a TPM matrix a zero is a real measurement (the gene is not expressed in that tissue), not a missing value. No false alarm was emitted.
