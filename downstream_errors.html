Resources whose own outputs carry the error

GTEx eQTL outputs & trans-eQTL / co-expression consumers

The GTEx project's released eQTL associations, and any study built on its trans-eQTL or co-expression calls.

literature-widecross-mapping

Up to ~75% of standard trans-eQTLs are alignment artifacts: reads from one gene mis-map to a sequence-similar gene. In GTEx, 75.14% of significant trans-eQTLs were cross-mappable and 90.8–97.3% of the top-1000 per tissue (vs 19.1–25.6% background). The consortium later added a cross-mappability filter; segmental-duplication mismapping still leaks through (2025).

Saha & Battle, F1000Research 2018; bioRxiv 2025.10.08.681258.

literature-widecontamination

Low-level RNA contamination (~40% of samples) gives pancreas genes (PRSS1, PNLIP, CELA3A, CLPS…) apparent eQTLs in tissues that don't express them — 31 of 72 tissues carried “inappropriate” eQTLs. A PRSS1/PNLIP eQTL in brain or blood is a same-sequencing-day artifact.

Halushka et al., Nature Communications 2020.

Reprocessed expression mirrors — UCSC Xena/Toil, Recount2/3, GDC, ARCHS4, MSKCC, Piccolo

Resources that re-align GTEx raw reads and republish gene-level expression for general use.

literature-widepipeline divergence

The same GTEx samples give discordant TPM across pipelines: 2,068 genes (12.36%) differ by >4-fold. The mirror you download materially changes gene-level conclusions; “the GTEx TPM” is not a single number. ARCHS4 specifically re-aligns GTEx and diverges from other sources.

Arora et al., Scientific Reports 2020 / UncertaintyRNA; ARCHS4.

confirmedaccess plumbing

recount3's GTEx bigwig download URLs were malformed (locate_url), breaking programmatic access.

LieberInstitute/recount3 #21.

GTEx Portal visualization — broadinstitute/gtex-viz

GTEx's own web visualization of gene- and transcript-level expression.

confirmedaggregation semantics

Gene-level median TPM is not the sum of its isoforms: for SELL in Whole Blood the gene value is 1349 TPM while six isoforms sum to 500.96 (2.7×). The gene value is a median (RNA-SeQC); transcripts are means (RSEM). The file is not composable from isoforms.

broadinstitute/gtex-viz #207.

eQTL Catalogue (EMBL-EBI)

Harmonised eQTL summary statistics across studies, including GTEx, for colocalization.

confirmeddata-quality

In release 3 (Oct 2020) the AC and AN fields were swapped; analyses that used AN to infer sample size had reduced power to detect colocalizations.

eQTL Catalogue release notes.

Application tools & packages that consume GTEx

GEPIA — cancer/normal expression web server

Compares TCGA tumor vs GTEx normal expression for differential analysis (heavily cited).

weakenedbatch confound

Tumor (TCGA) vs normal (GTEx) is a nested design. GEPIA reprocesses both through UCSC Xena/Toil to mitigate the batch confound — so it is a residual effect, not the uncorrectable one first stated — but a TCGA-vs-GTEx batch component persists, so calls still warrant independent validation.

GEPIA; residual-batch caveat per Arora et al. 2020.

Signac — single-cell ATAC-seq (Stuart/Satija lab)

Companion to Seurat; GeneActivity() maps accessibility to genes.

confirmedduplicate IDs

Pseudoautosomal (_PAR_Y) genes carry a duplicate ID, so GeneActivity() produced a matrix with 36,588 rows but 36,559 unique names — 29 PAR genes duplicated, fanning out downstream sums/joins.

timoast/signac #521.

PheKnowLator — biomedical knowledge graph

Ingests GTEx median TPM to build gene–tissue expression edges.

confirmednon-unique key

Because a gene symbol maps to many gene IDs (in this matrix 199 symbols → multiple Ensembl IDs), the pipeline must deduplicate symbols by hand (e.g. HLA-DQA2) or emit colliding/duplicated edges.

callahantiff/PheKnowLator.

grex — GTEx gene-ID mapping R package

Maps GTEx Ensembl IDs to symbols/Entrez for annotation.

confirmednon-unique key

Hundreds of cases where one Ensembl ID maps to multiple Entrez IDs; grex keeps only the first and warns its mapping “may not be 100% accurate” — a documented, lossy collapse.

grex vignette.

SuSiE GTEx preprocessing (Stephens Lab)

Prepares GTEx genes/associations for statistical fine-mapping.

unguardedduplicate IDs

Gene extraction is sort -u by gene ID only, with no chromosome-aware handling, so _PAR_Y duplicate IDs pass through unfiltered.

SuSiE GTEx preprocessing.

Method & transparency

Verification. Each error was checked against its primary source and the numbers quoted from the original. Quantitative claims (75.14%; 90.8–97.3%; 12.36% / >4-fold; 31 of 72 tissues; 1349 vs 501 TPM; 36,588 vs 36,559 rows; AC/AN swap) reproduce exactly.

Rejected. A retracted lung-adenocarcinoma expression-mining paper was investigated and excluded: its retraction cites systematic publication-process manipulation and peer-review integrity, not a GTEx/TCGA data error.

Weakened. GEPIA's confound was downgraded from “uncorrectable” to “residual” after confirming its uniform reprocessing; ARCHS4 is listed under the reprocessed mirrors as corroborating. The shared pattern across every resource: it treated a GTEx identifier or value as cleaner than it is — duplicate IDs, a symbol as a key, cross-mapping reads, contamination, pipeline-variant values, or a median mistaken for a sum.
